# 30. Bun

Date: 2026-05-30

## Status

Accepted

## Context

Meadow used npm for JavaScript dependency installation, tests, and build-time packaging. Bun provides faster installs and test execution, and the related `dc-api-v2` project has already moved its JavaScript tooling to Bun.

Supersedes [29. npm](0029-npm.md)

## Decision

Use `bun` instead of `npm` for JavaScript dependency installation, frontend test execution, and Lambda packaging commands. Continue using Node.js as the deployed JavaScript runtime where AWS Lambda or application scripts require it.

## Consequences

JavaScript package roots use `bun.lock` as the committed lockfile. Developers and CI need Bun installed at the version pinned in `.tool-versions`.
